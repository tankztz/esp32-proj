Core

  • MCU: ESP32-D0WDQ6, revision v1.0
  • Wi-Fi, Bluetooth, dual core, 240MHz
  • Flash: 16MB, manufacturer c8, device 4018
  • PSRAM: 64Mbit / 8MB

Display

  • ILI9341-compatible SPI TFT
  • Draw as 320x240

  • SPI clock tested at 40MHz

Rebuild Roadmap

A practical sequence for recreating the original Rubik Cube-1 firmware in ESP-IDF.

1. UI Shell

Build a landscape 320x240 home screen, status bar, theme toggle, and menu tiles.

Display proven

2. Touch

Read the FT6X36-compatible controller at 0x38, calibrate coordinates, and wire a home gesture.

3. Settings

Backlight brightness, time setting, Wi-Fi mode selection, and persistent preferences.

4. Wi-Fi

Off, STA scan, SoftAP, and ESPTouch/SmartConfig like the original firmware strings indicate.

5. Files

Mount SPI SD on CS GPIO4, list root files, and open simple text entries.

6. Sensors

Show MPU6886 acceleration, gyro, and temperature. Probe GPIO5 accessory behavior separately.
